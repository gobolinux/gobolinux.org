Documentation

This website is the primary source of information about GoboLinux.

Starting points

Things you'll probably want to read first:

  • GoboLinux at a glance - a visual overview of what makes this distribution unique.
  • Frequently Asked Questions - the first question is "what the heck is GoboLinux", so, if you're still wondering... check it out!
  • GoboLinux Wiki - our community wiki, with lots of documentation for the GoboLinux tools.

Articles

Informal pieces written by GoboLinux developers, which hopefully will shine some light on the spirit of the project and on what goes on the dev's heads. :) Most of these are just emails sent to gobo-l which were considered worth keeping.

  • The Unix tree rethought: an introduction to GoboLinux - the article that originally presented GoboLinux to the world. Published on Kuro5hin.
  • Myths and misconceptions about the design of GoboLinux. Makes for good bed-time reading. Also in printer-friendly PDF.
  • Compile - the GoboLinux compilation system - an introduction to our build tool
  • The ideas behind Compile - the ideas on which Compile and the Recipes tree are based on.
  • GoboHide: surviving aside the legacy tree - an explanation on how GoboHide was developed and how it works.
  • Installing programs by hand - how to install programs "by hand" in GoboLinux without using Compile.
  • Release History - a page collecting the release announcements for all past releases of GoboLinux

Advanced guides

For tweaking the distro beyond normal use!

  • Tweaking the GoboLinux LiveCD: a tutorial - information on how to use the GoboLinux tools to make a customized version of the GoboLinux Live CD.
